# 🎬 Netflix GPT

Netflix GPT is an AI-powered movie recommendation web application inspired by Netflix.  
It allows users to browse movies, watch trailers, and get intelligent movie suggestions using Gemini GPT and TMDB APIs.

# 📦 Installation

## Clone Repository

```bash
git clone https://github.com/your-username/netflix-gpt.git
```

## Navigate to Project

```bash
cd netflix-gpt
```

## Install Dependencies

```bash
npm install
```

## Start Development Server

```bash
npm run dev
```

or

````bash
npm run dev

# 🔥 Firebase Setup

1. Create a project on Firebase
2. Enable Authentication
3. Enable Email/Password Sign In
4. Copy Firebase Config Keys
5. Add Keys to `.env`

---

# 🎬 TMDB API Setup

1. Create account on TMDB
2. Generate Access Token
3. Add TMDB API Key in `.env`

---
````
